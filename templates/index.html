How to Use DFS

1. Start Services

Run coordinator and all storage nodes from your terminal.

python coordinator.py

2. Upload Files

Split and replicate files using the client command.

python client.py upload test.txt

3. Monitor Health

Use this dashboard to watch nodes and stored files in real-time.

http://localhost:5150
